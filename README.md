# TimeAnomaly-detect
# Прогнозирование и обнаружение аномалий во временных рядах

## Цель работы
Разработка учебного прототипа системы, выполняющей:
1. Прогнозирование будущих значений временного ряда.
2. Обнаружение аномальных значений (перегрев, сбои датчиков).

## Датасет
**Numenta Anomaly Benchmark (NAB)** — `machine_temperature_system_failure.csv`  
Источник: https://www.kaggle.com/datasets/boltzmannbrain/nab

Данные загружаются автоматически из репозитория NAB, скачивать вручную не нужно.

## Предметная область
Промышленная телеметрия, мониторинг состояния оборудования.  
Аномалиями считаются резкие пики температуры, выходящие за пределы нормального рабочего цикла.

## Методы
- **Прогнозирование:** скользящее среднее (базовая), SARIMA (дополнительная).
- **Обнаружение аномалий:** Z-Score, IQR, Isolation Forest.

## Результаты

### Прогнозирование
| Модель | MAE | RMSE | MAPE |
|--------|-----|------|------|
| Скользящее среднее | 1.6249 | 3.1450 | 2.05% |
| SARIMA | 13.8509 | 23.5163 | 28.72% |

**Лучшая модель:** Скользящее среднее.

### Обнаружение аномалий
| Метод | Аномалий | Процент |
|-------|----------|---------|
| Z-Score | 68 | 1.50% |
| IQR | 274 | 6.04% |
| Isolation Forest | 227 | 5.00% |
